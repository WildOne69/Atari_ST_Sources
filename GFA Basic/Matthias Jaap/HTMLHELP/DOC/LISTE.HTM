Wählen sie die Kriterien, die in der Liste aufgeführt werden sollen und klicken sie auf OK.
HTML-Help wird nun eine HTML-Datei erzeugen, die die Daten der Hotlist enthält. Dateiname und URL-Adresse werden dabei jeweils als Verweise erzeugt, d.h. man kann die Tabelle auch als so eine Art Inhaltsverzeichnis in CAB verwenden.

Besonderheiten
Ist 'Titel' selektiert, 'Dateiname' allerdings nicht, so wird HTML-Help den Titel als Verweis definieren. Klickt man in CAB auf den Titel, wird dann die entsprechende Datei geladen.
