Dead Hackers Society Falcon Demosystem v15

This demosystem is intended for use with
Falcon computers and is free to use. 


For detailed information about using this 
demo sequencer, please refer to the first
issue of the diskmag 'Alive'. It can be found
on Alives homepage: http://alive.atari.org/

If you want this and that added to the demosys,
then you're free to modify it to your wishes :)
In it's current state, it's made for the needs
of DHS, which will not be suitable for everyone.
Grab the bits and pieces you need!

If you have further questions, don't hestitate
to contact me.



Keep releasing new stuff!
