Whirlgif example
----------------

If you have mupfel [a shell] installed, doubleclik on example.mup. 
Whirlgif will merge all gif into one single animated gif.

Or enter in a shell:

 whirlgif -o world.gif -loop -trans -0,0,0 -time 10 -i include.txt

From the desktop:

 Copy whirlgif.ttp into example folder.
 Doubleclick whirlgif.ttp
 Enter "-o world.gif -loop -trans -0,0,0 -time 10 -i include.txt"
 Delete whirlgif.ttp

Arguments
---------

 -o world.gif     - create world.gif, an animated gif
 -loop            - make infinite loop
 -trans -0,0,0    - Make RGB #000000 as trans color. (Black)
 -time 10         - 10/100 sec (=1/10 s) pause between each frame
 -i include.txt   - a file with all gif listed

Start Cab and load world.gif. Note! If you are reloading the image, 
press Shift while clicking on reload button.

The images of the world was created with xearth.
